Grayscale

A free, responsive, one page Bootstrap theme.
Adapted to integrate Social Login with AngularJS.

About Grayscale Theme

SSINCO's customization of Grayscale integrates Satellizer, an authentication module for AngularJS, you can use it as starting point for Social Login Authentication and Authorization. For server-side you can use Satellizer's examples or our customization (coming soon).

Grayscale is a free Bootstrap 3 theme created by Start Bootstrap. The theme is open source, and you can use it for any purpose, personal or commercial.

This theme features stock photos by Gratisography and Magdeleine.
